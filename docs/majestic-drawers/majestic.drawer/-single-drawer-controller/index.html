SingleDrawerController

interface SingleDrawerController

Controls all drawers owned by a single-drawer host.

Use this with InlineDrawerHost or OverlayDrawerHost when a screen only needs one drawer. Calling the parameterless operations affects the hosted drawer. Passing a ratio or span opens or toggles the drawer with an override size for that state change.
