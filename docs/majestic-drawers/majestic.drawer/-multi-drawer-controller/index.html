MultiDrawerController

interface MultiDrawerController : SingleDrawerController

Controls a DrawerHost that can contain multiple drawers.

Methods that accept a Drawer target one concrete drawer instance. Methods that accept Any treat the value as a key and affect every drawer registered with that key. Parameterless operations come from SingleDrawerController and affect every drawer known to the controller.
